how a hash table work?

A hash table uses a hash function to compute an index, also called a hash code, into an array of slots, from where you can accuses the data. The hash function has to be sufficient enough to prevent collisions. Collisions occurs when the next number to be proceeded gets put into a slot of the array where there is already a key value pair placed due to the size of the array. If there is to many collision it effects the time complexity drastically when retrieving values out of a hash table
