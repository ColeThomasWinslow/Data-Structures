There Are Two Types of Arrays

Static Array (Sometimes Used In Javascript with Array Constructor )

If the array is static, when you declare your variable you have to specific the amount of items that will be in your array so that your OS will have enough space in memory. Also arrays are stored in memory contiguously, meaning the memory spaces will always be stored back to back wether its dynamic or static.

If the Array is Dynamic, it means that when it is declared you don't have to specify the amount of the starting array and within memory it knows the arrays length so it can allocate more memory as it continues to grow and need more space.

Complexity Analysis

Get An Item from an Array: O(1)

Setting Item from an Array: O(1)

Initializing An Array: O(N)

Traversing An Array: O(N)

Copying An Array:O(N) Space

Adding New Items: Depends on Static Or Dynamic

Static O(N)

You need to copy the whole array then create a new array with the newly added data

Dynamic O(1)

With Dynamic arrays you already have allocated extra space in memory when you created it so adding inst very costly
